# 🏥 Hospital Management System

A **Java-based Hospital Management System** developed to simplify hospital administration through an efficient database management system. The application is built using **Java Swing** for the graphical user interface, **JDBC** for database connectivity, and **MySQL** as the backend database.

The system provides a centralized platform for managing patient records, employee information, room allocation, ambulance services, departments, and patient discharge. It minimizes manual paperwork, improves data organization, and enhances the overall efficiency of hospital operations.

---

## ✨ Features

- 👤 Patient Registration & Information Management
- ✏️ Update Patient Details
- 🛏️ Room Allocation & Availability
- 🚑 Ambulance Management
- 👨‍⚕️ Employee Management
- 🏢 Department Management
- 📤 Patient Discharge
- 🔍 Search & Record Management
- 💾 MySQL Database Integration
- 🖥️ Interactive Java Swing GUI

## ⚙️ Installation

1. Clone this repository.
2. Open the project in IntelliJ IDEA, Eclipse, or NetBeans.
3. Create a MySQL database.
4. Import the provided SQL file.
5. Update the database credentials in the connection class.
6. Run the project.
